CLASSIFICAÇÃO DE ROCHAS GERADORAS DE PETRÓLEO A PARTIR DE
DADOS HIPERESPECTRAIS E MACHINE LEARNING

Esta aplicação permite ao usuário duas opções:

1. Predição: classificar o seu dado hiperespectral de acordo com classes pré-definidas a partir de um modelo já treinado e validado;

Informações sobre este modelo: algoritmo de classificação SVM, 3 classes (alta (classe 0), média (1) e baixa (2) fração orgânica) e 4 features com feições espectrais características (f1, f2, f3 e f4). Acurácia do modelo = 0,90.

2. Classificação: Utilizar sua própria base de dados para treinar e validar um novo modelo de classificação por SVM, podendo optar por quais classes e features serão utilizadas.

(É possível depois classificar novos dados a partir deste seu modelo.)

Nota: Se esta opção (Classificação) for utilizada, você deve ter um conjunto de dados maior para que o treinamento e validação do algoritmo tenham informações suficientes para gerar modelos consistentes e que possam ser generalizados.

Para que a classificação dos seus dados hiperespectrais ocorra corretamente, fique atento aos seguintes aspectos:

1. O arquivo com os dados que serão preditos ou classificados deve estar na extensão .csv e atender os seguintes requisitos: (1) utilizar como separador entre as colunas ponto e vírgula (;) e como separador decimal ponto (.); (2) conter dados de amostras nas linhas e espectrais nas colunas; (3) os rótulos das colunas devem ser 'Classe' (válido apenas para função de classificação), 'Amostra' e as bandas espectrais de interesse.

2. Insira o arquivo que você deseja utilizar na pasta de trabalho “app/input”, nomeado por “predict.csv” ou “classification.csv”, dependendo de qual função será utilizada na aplicação.

3. Somente após concluir as etapas 1 e 2, clique “Predizer classes com modelo SVM existente” ou em “Gerar novo modelo de classificação SVM”.
